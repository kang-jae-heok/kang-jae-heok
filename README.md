## About Me

### 코드로 마케팅 하는 iOS 앱 개발자 강재혁입니다 :)

‘앱’ 은 실용적이어야 합니다. 현재의 사람들이 가장 필요로 하고, 좋아하는 것이 무엇인지 늘 주시해야 합니다. 그래야만 가장 널리, 오래 사용될 실용적인 앱을 만들 수 있기 때문입니다.
